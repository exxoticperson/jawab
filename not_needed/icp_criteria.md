# Jawab ICP & Target Clinic Profile

This document outlines the Ideal Customer Profile (ICP) for Jawab and the criteria used to score and rank leads.

## Target Clinic Profile

| Attribute | Criteria |
|-----------|----------|
| **Type** | Private dental clinic, owner-operated or manager-accessible |
| **Size** | 1–3 locations |
| **Specialty** | Cosmetic, implant, or aesthetic dentistry (highest patient LTV, most competitive, most urgent about missed calls) |
| **City** | Dubai (Priority 1) → Sharjah (Priority 2) → Abu Dhabi (Priority 3) |
| **Decision maker** | Owner-dentist or clinic manager |

## Disqualifiers

Do NOT include clinics that meet any of these criteria:
- Large chains or corporate hospitals
- Clinics with no online presence
- Dedicated call centers
- Clinics outside Dubai, Sharjah, or Abu Dhabi
- Hospital dental departments or university clinics

## ICP Scoring System (0-3 Scale)

Score each clinic based on the following criteria. The maximum score is 3.

| Criteria | Points | Description |
|----------|--------|-------------|
| **Specialty match** | 0 or 1 | 1 point if the clinic specializes in cosmetic, implant, or aesthetic dentistry. |
| **Digital presence** | 0 or 1 | 1 point if the clinic has an active Instagram (posts in last 30 days) AND a website. |
| **Pain signal** | 0 or 1 | 1 point if at least one Google review mentions phone/booking complaints. |

## Tiering System

After scoring, sort the full list by ICP score descending, then by Google review count descending. Create three output tiers:

- **Tier A (Top 30)**: ICP score 3, OR ICP score 2 with 100+ Google reviews. These receive high-personalization outreach.
- **Tier B (Next 70)**: ICP score 2. These receive standard outreach.
- **Tier C (Remaining)**: ICP score 0-1. These serve as a backup pipeline; no immediate outreach.

## Pain Signal Keywords

When scanning Google reviews, look for these keywords (in both English and Arabic) that indicate phone or booking issues:

- "couldn't reach" / "ما قدرت أتواصل"
- "no answer" / "ما حد رد"
- "didn't call back" / "ما اتصلوا"
- "hard to book" / "صعب الحجز"
- "waited long time for response" / "انتظرت وايد"
- "receptionist" / "ريسبشن"
- "phone" / "تلفون"
- "WhatsApp" / "واتساب"
