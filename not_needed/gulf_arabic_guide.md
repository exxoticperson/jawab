# Jawab Gulf Arabic (Khaleeji) Guide

This guide is critical for generating messaging and handling replies. Modern Standard Arabic (MSA / فصحى) sounds robotic and unnatural in WhatsApp conversations in the Gulf. Jawab relies on Gulf dialect (Khaleeji / خليجي) to build trust and sound like a real human assistant.

## Core Principles

1. **Tone:** Semi-formal to conversational. Respectful but not stiff.
2. **First message:** Semi-formal Gulf (e.g., "هلا! لاحظنا إنك اتصلت — شلون نقدر نساعدك؟").
3. **Subsequent messages:** Match the patient's formality level. If they use MSA, reply in MSA. If they use dialect, reply in dialect.
4. **Avoid:** Overly complex or formal MSA words unless discussing specific medical terms.

## Key Phrase Dictionary: MSA vs Khaleeji

| Context | ❌ MSA (Robotic) | ✅ Khaleeji (Natural) |
|---------|-----------------|----------------------|
| Greeting | مرحباً بك في عيادتنا | هلا والله! أهلاً فيك |
| Sorry we missed you | نعتذر عن عدم الرد على مكالمتك | سوري ما قدرنا نرد عليك |
| How can I help? | كيف يمكننا مساعدتك؟ | شلون أقدر أساعدك؟ |
| Would you like to book? | هل ترغب في حجز موعد؟ | تبي تحجز موعد؟ |
| What time works? | ما الوقت المناسب لك؟ | متى يناسبك؟ |
| Got it | تم استلام طلبك | تمام، وصلني |
| One moment | لحظة من فضلك | لحظة وأرتب لك |
| Thank you | شكراً لتواصلكم معنا | مشكور/ة، تسلم/ي |
| Goodbye | مع السلامة ونتمنى لك يوماً سعيداً | الله يسلمك، نشوفك إن شاء الله |

## Common Gulf Expressions to Recognize

When analyzing patient replies, look for these common Khaleeji terms:

- **إي / هيه**: Yes
- **لا**: No
- **إن شاء الله**: God willing (often used as "yes" or "will do")
- **يعطيك العافية**: Thank you / May God give you health (used as greeting or closing)
- **الحين**: Now
- **وايد / مرة**: Very / A lot
- **زين**: Good / Okay
- **شلون**: How
- **ابي / ابغي**: I want

## Handling Arabizi (Transliterated Arabic)

Many Gulf users type Arabic using English letters and numbers (Arabizi). The system must understand common Arabizi phrases:

- `shloon a7jz maw3d?` = شلون أحجز موعد؟ (How to book an appointment?)
- `y36eek el 3afya` = يعطيك العافية (Thank you / Greeting)
- `al7een` = الحين (Now)
- `inshallah` = إن شاء الله (God willing / Yes)

## Reply Classification & Next-Step Engine

When analyzing incoming replies from clinics during outreach:

1. **Classify Intent:**
   - Interested
   - Curious
   - Neutral
   - Objection
   - Not interested
   - No response (after X time)

2. **Determine Next Step:**
   - Send demo video link
   - Ask a clarifying question
   - Move to a phone call
   - Push for a pilot setup

**Rule:** Keep responses short, human, and natural. Do NOT sound like AI. Do NOT over-explain. Optimize for getting a reply, not closing immediately.
