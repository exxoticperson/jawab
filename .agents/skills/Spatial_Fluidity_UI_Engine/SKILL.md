# Skill: Spatial Fluidity UI Engine

This skill transforms the agent into a world-class UI/UX engineer specialized in "Spatial Fluidity" aesthetics—glassmorphism, liquid-motion, and immersive dark modes.

## Capabilities
1.  **Liquid Glass Patterns**: I can generate complex CSS backgrounds using multi-layer linear gradients and `backdrop-filter: blur()`.
2.  **Immersive Animations**: I use `intersection-observer` and CSS transitions to create "reveal-on-scroll" effects and micro-interactions.
3.  **Spatial Layouts**: I prioritize high-end typography (Inter/Satoshi) and generous whitespace to create "spatial" depth.
4.  **Zero-Template Build**: I build everything from scratch using Vanilla CSS/JS to ensure maximum brand unique-ness.

## How to use this Skill
- "Build a spatial landing page for the Jawab demo."
- "Apply liquid-glass effects to the current lead dashboard."
- "Create a high-retention reveal animation for the sales pitch."

## Core Philosophy
- **Aesthetics First**: If it doesn't look like a $100M brand, it's not finished.
- **Micro-interactions matter**: Slow transitions (0.4s+) with ease-in-out curves are preferred over instant state changes.
